# Exploration Directory

This directory contains ad-hoc analysis scripts, test files, and exploration utilities.

## Purpose

- **Ad-hoc analysis**: Quick exploration scripts for data analysis
- **Testing**: Test scripts for validating utilities
- **Debugging**: Scripts for troubleshooting issues
- **Experimentation**: Trying out new approaches

## Guidelines

- Files in this directory are typically temporary
- Can be deleted after use or when no longer needed
- Keep descriptive names for future reference
- Document complex exploration logic in comments

## Examples

- `test_*.py` - Test scripts
- `check_*.py` - Validation scripts
- `verify_*.py` - Verification scripts
- `notebook_*.py` - Test notebooks
